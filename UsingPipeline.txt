BITS (Background Intelligent Transfer Service) It is essential for background downloads and 
uploads and can be scheduled to work on idle time. It can be used to sync OneDrive and Microsoft defender

Get-Process | Stop-Process -WhatIf
PowerShell prints a simulation of the actions instead of executing it
It prevents accidental deletion or modification. It lets you validate the logic before running a risky script.
It helps you understand how cmdlets work.

*Foreach Loop
A foreachloop goes through the list of objects and performs the same action for every object in an array.It will iterate over the list.
There are multiple ways of using this loop
1. foreach statement
2. Foreach-Object cmdlet
3. Foreach() method

Get-Process | Foreach-Object{$PSItem}
$PSItem automatic variable that suggest the current object in pipline. Similar to $_
